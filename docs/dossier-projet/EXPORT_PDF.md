# Export PDF

Le dossier est prevu pour etre exporte depuis :

- `docs/dossier-projet/dossier-projet.html`

## Methode recommandee

1. Ouvrir le fichier HTML dans un navigateur.
2. Lancer l'impression.
3. Choisir `Enregistrer au format PDF`.
4. Utiliser le format `A4`.
5. Garder les arrieres-plans actifs si le navigateur le propose.
