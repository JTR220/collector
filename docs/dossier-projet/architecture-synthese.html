Synthese Architecture - collector.shop

Fiche courte destinee a la soutenance, a la revue technique ou a une lecture rapide du systeme.

Vue d'ensemble

Utilisateur | v collector-front |----> auth-service ------------> PostgreSQL collector |----> catalog-service ---------> PostgreSQL collector |----> price-tracker-service ---> PostgreSQL collector_price |----> notification-service ---> PostgreSQL collector_notifications `----> WebSocket notification-service catalog-service -- price.updated / order.* --> RabbitMQ RabbitMQ -------> price-tracker-service RabbitMQ -------> notification-service price-tracker-service -- fraud.alert --> RabbitMQ

Domaines

Frontend

  • SvelteKit + TypeScript
  • catalogue, dashboard, admin, vente, messages
  • consomme REST et WebSocket

Auth

  • creation de compte et login
  • JWT partage avec les autres services
  • routes admin pour la moderation utilisateurs

Catalogue

  • articles, images, categories
  • achat, commandes, wishlist, reviews
  • publication des evenements metier

Prix et notifications

  • historique des prix et anti-fraude
  • alertes admin
  • notifications temps reel et messagerie

Flux cle

1. Login Frontend -> auth-service -> JWT 2. Mise a jour de prix Frontend -> catalog-service -> RabbitMQ -> price-tracker-service price-tracker-service -> collector_price price-tracker-service -> RabbitMQ -> notification-service -> WebSocket 3. Achat Frontend -> catalog-service -> order.created -> RabbitMQ -> notification-service

Execution et livraison

Local : Docker Compose -> postgres, rabbitmq, mailhog, services, front, prometheus, grafana Cible : GitHub Actions -> scans / build / SBOM / cosign -> commit des digests dans infra/k8s/overlays -> Argo CD synchronise le cluster -> prod promue par PR puis sync manuel

Points differenciants

  • architecture event-driven sur un sujet fonctionnel concret
  • separation nette entre identite, catalogue, prix et notifications
  • approche DevSecOps deja visible dans le depot
  • cible GitOps coherente avec un environnement de staging et de prod
